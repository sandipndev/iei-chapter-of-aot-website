Welcome to IEI Student's Chapter of Academy Of Technology

We are a community of university students who code for fun and organize cultural activities. We think of an idea, form a team and get down to planning it out. We create a backend in the Cloud, take inputs from IOT devices and develop a cool looking Web App. Then we get down on Flutter and generate an awesome app. Finally, we sprinkle a pinch of Machine Learning to it and make it intelligent. We do tech...

There are so many domains to explore in today's world. But the reality is that, when making a production-level application, all of the domains come together to make the app cool!

Hence, when we work on an idea, we start designing the back-end on the Cloud. The front-end team starts to design using Figma or Design XD. Then they write the html-scss code and use great Js frameworks like Vue. Finally the backend and this web app integrates properly. There is someone looking after the database. When this system was being built, there was a team dedicated to make an Android app using Flutter or Kotlin.

Oh, and when it's almost ready, we sprinkle Machine Learning in every possible place and fine tune the knobs before it goes to production.
